# Ymir CLI

The [Ymir][1] command-line tool.

## Requirements

 * PHP >= 7.2.5

## Installation

Install the Ymir CLI in your project using composer:

```
$ composer require ymirapp/cli
```

Or globally:

```
$ composer global require ymirapp/cli
```

## Contributing

Install dependencies using composer:

```console
$ composer install
```
